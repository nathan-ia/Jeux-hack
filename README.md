# Documentation Web Interactive

Ce guide est une ressource personnelle pour maîtriser la création de sites web. Il regroupe mes notes sur la structure, le design et l'interactivité.

## Utilisation

Chaque concept est expliqué avec :
- **La Théorie** : Pourquoi et comment utiliser chaque balise.
- **Le Code** : Des exemples concrets à copier et coller.
- **Le Rendu** : Une zone "Live Sample" pour voir le résultat en direct.

## Structure

- **Introduction** : Présentation du guide.
- **Structure HTML** : Balise de base pour commencer une page HTML.
- **index.html** : Sera mis a jour a chaque nouvelle découverte de la part de son développeur.
