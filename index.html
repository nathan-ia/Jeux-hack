Introduction

Documentation Web Interactive

Ce guide est une ressource personnelle pour maîtriser la création de sites web. Il regroupe mes notes sur la structure, le design et l'interactivité.

Ici, chaque concept est expliqué avec :

  • La Théorie : Pourquoi et comment utiliser chaque balise.
  • Le Code : Des exemples concrets à copier et coller.
  • Le Rendu : Une zone "Live Sample" pour voir le résultat en direct.

Utilise le menu à gauche pour naviguer entre les différentes technologies (HTML, CSS, JS) et voir comment elles travaillent ensemble pour créer un site complet.

Structure de base HTMLStandard

Pour bien commencer une page HTML, on doit commencer par la déclaration DOCTYPE. Voici un exemple :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma Première Page HTML</title>
</head>
<body>
</body>
</html>

La balise <title> sert à mettre le titre de la page dans l'onglet du navigateur.

Les titresStandard

Le <h1> sert à faire un titre principal, le h2 sert à faire un titre secondaire, et le <h3> un titre de troisième niveau ect jusque au <h6>.

Exemple :

<h1 class="titre">Hello World</h1>

On a mis <h1> pour dire que c'est le titre principal de la page.

Les DivStandard

Les div servent à regrouper des éléments dans le HTML. On doit metre une class ou un id dedans pour pouvoir les selectionner et les modifier avec CSS ou avec le JavaScript(JS).

Exemple :

<div class="conteneur">
    <h1>Hello World</h1>
    <p>Bonjour tout le monde !</p>
</div>

Les classesStandard

Les classes servent à regrouper des éléments pour les styliser avec CSS ou les sélectionner avec JavaScript.

Exemple :

<h1 class="titre">Hello World</h1>

On a mis class="titre" pour dire que c'est la classe titre.

On pourras le styliser avec CSS avec .titre ou le sélectionner avec JavaScript avec document.querySelector('.titre').

Les IDStandard

Les id servent à identifier un élément unique pour le styliser avec CSS ou le sélectionner avec JavaScript. Comme les classes, on met un nom dedans.

Exemple :

<h1 id="titre">Hello World</h1>

On a mis id="titre" pour dire que c'est l'id titre. On pourras le styliser avec CSS avec #titre ou le sélectionner avec JavaScript avec document.querySelector('#titre').

Les boutonsStandard

Pour rendre un bouton cliquable, il faut mettre la balise <button> avec un id="" dedans. C'est un nom que tu choisis toi-même comme pour les classes.

On peut modifier son apparence avec CSS et déclencher des actions avec JavaScript.

<button id="monBtn">Cliquez ici</button>

AudioStandard

La balise <audio> permet d'insérer un fichier audio sur la page.

Exemple :

<audio src="mon-audio.mp3" controls></audio>

On peut ajouter la balise controls pour afficher les contrôles de lecture.

CSSStandard

Le CSS permet de modifier l'apparence de la page (taille, couleur, police) pour la rendre plus attrayante.

Il faut mettre la balise <style> dans la balise <head> ou lier un fichier .css.

<link rel="stylesheet" href="nomdufichier.css">

Exemple : background-color est la couleur de fond.

Le TableauStandard

S'écrit avec la balise <table> et permet de créer un tableau de données.

Exemple :

<table>
                <tr>
                <th>Nom</th>
                <th>Prénom</th>
                </tr>
                <tr>
                <td>Dupont</td>
                <td>Jean</td>
                </tr>
                </table>

La barre de saisieStandard

La balise <input> permet de créer une zone de saisie. On peut récupérer sa valeur avec JavaScript.

Voici un exemple de code :

<input type="text" id="maSaisieDoc" placeholder="Saisissez votre nom">

Insérer des images Standard

La balise <img> permet d'afficher des images sur ton site. C'est une balise orpheline (elle n'a pas de fermeture).

Il existe deux façons de charger une image :
  • Lien Local : L'image est enregistrée dans ton dossier (ex: image.jpg).
  • Lien Internet (URL) : L'image est hébergée sur un autre site (ex: https://...).

1. Image locale (dans ton dossier)

C'est la méthode la plus sûre pour que ton site fonctionne toujours.

<img src="mon-image.png" alt="Un chat qui dort" width="300">

2. Image depuis Internet (URL)

On copie simplement l'adresse de l'image trouvée sur le web.

<img src="https://exemple.com/photo.jpg" alt="Paysage de montagne">
Rappel important :
  • L'attribut src est la source (le lien).
  • L'attribut alt est obligatoire pour l'accessibilité (si l'image ne charge pas).

Le LienStandard

La balise <a> permet de créer un lien vers une autre page ou vers une autre section de la même page.

Exemple :

<a href="https://www.google.com">Google</a>

Clique sur le lien pour voir le résultat :

Mon site

La listeStandard

Il existe deux types de liste :

  • La liste à puces avec <ul>
  • La liste à numéros avec <ol>

Exemple de liste à puces :

<ul>
    <li>Pomme</li>
    <li>Poire</li>
    <li>Banane</li>
</ul>

Exemple de liste à numéros :

<ol>
    <li>1</li>
    <li>2</li>
    <li>3</li>
</ol>

Exemple :

  • Pomme
  • Poire
  • Banane
  1. Un
  2. Deux
  3. Trois

Le JavaScript (JS) Dynamique

Si le HTML est le squelette et le CSS les vêtements, le JavaScript est le muscle qui permet de bouger.

Rôle du JS : Il sert à rendre la page réactive. Par exemple :
  • Cacher ou montrer un menu.
  • Calculer des nombres.
  • Changer le texte d'un titre quand on clique dessus.
